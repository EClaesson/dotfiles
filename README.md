# Dotfiles
These are my personal dotfiles for EndeavourOS with KDE Plasma 6.

You are free to use whatever you want from here, but note that there are hardcoded references to my name and email in some files.

# Install
```sh
yay -Sy chezmoi
chezmoi init --apply --verbose git@github.com:EClaesson/dotfiles.git
~/scripts/setup_endeavour.sh
```
